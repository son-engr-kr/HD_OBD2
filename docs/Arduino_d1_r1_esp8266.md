## ESP8266
- ESP8266(또는 ESP8266EX)은 Wi-Fi 무선 통신 기능을 탑재한 저전력 마이크로컨트롤러
- ESP8266모듈을 사용 시 하나의 아두이노에 mcp2515,esp8266모듈 동시에 SPI통신 안됨
- ESP8266이 내장된 아두이노 우노(WeMos) D1 R1 또는 아두이노 우노(WeMos) D1 R2
